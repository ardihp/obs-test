## Run Locally

Clone the project

```bash
  git clone https://github.com/ardihp/obs-test.git
```

Go to the project directory

```bash
  cd obs-test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

To test the code

```bash
  npm run test
```
